Resume-template

It is a latex template which one can use to prepare their resume.

download .ZIPdownload .TGZ

This template can be used to make your resume in proper format and resulted output will be pdf file. Change the contents as per your need. of resume.tex file into your own contents. To run the resume.tex file follow following procedure: 1.) Download the files from above download link. 2.) Extract the template folder. 3.) Open the terminal. 4.) Enter into the directory which contains the resume.tex file. 5.) Run command: $pdflatex resume.tex Now you have a resume.pdf file. Enjoy it!!!
